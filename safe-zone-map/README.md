# نقشه نقاط امن 🗺️

یک برنامه تحت وب کاملاً آفلاین برای مدیریت و ذخیره موقعیت‌های مهم مانند بیمارستان‌ها، داروخانه‌ها و پناهگاه‌ها.

## ویژگی‌ها ✨

### قابلیت‌های اصلی
- ✅ **کاملاً آفلاین**: بدون نیاز به سرور یا اینترنت
- 💾 **ذخیره‌سازی محلی**: تمام داده‌ها در مرورگر ذخیره می‌شوند (LocalStorage)
- 📤 **خروجی JSON**: امکان ذخیره داده‌ها به صورت فایل JSON
- 📥 **ورودی JSON**: بازیابی و وارد کردن داده‌های ذخیره شده
- 🔍 **جستجو و فیلتر**: جستجو در نام، آدرس و توضیحات + فیلتر بر اساس نوع
- 📊 **آمار لحظه‌ای**: نمایش تعداد هر نوع مکان
- 🎨 **طراحی زیبا**: رابط کاربری مدرن و کاربرپسند

### انواع نقاط قابل ثبت
- 🏥 **بیمارستان**
- 💊 **داروخانه**
- 🏠 **پناهگاه**
- 🚑 **اورژانس**

- 👮 **پلیس**

### اطلاعات قابل ذخیره برای هر نقطه

## نصب و راه‌اندازی 🚀

### پیش‌نیازها
- یک مرورگر مدرن (Chrome، Firefox، Safari، Edge)
- هیچ نصب یا وابستگی اضافی نیاز نیست

### مراحل نصب

#### روش 1: استفاده مستقیم
1. فایل‌ها را دانلود کنید
2. فایل `index.html` را با مرورگر خود باز کنید
3. برنامه آماده استفاده است!

#### روش 2: استفاده با سرور محلی (اختیاری)
اگر می‌خواهید از یک سرور محلی استفاده کنید:

```powershell
# با Python
python -m http.server 8000

# یا با Node.js (نیاز به نصب http-server)
npx http-server -p 8000
```

سپس به `http://localhost:8000` بروید.

## نصب فونت Vazirmatn 📝

برنامه آماده استفاده از فونت Vazirmatn است. برای نصب فونت:

1. فایل‌های فونت Vazirmatn را دانلود کنید:
   - `Vazirmatn-Regular.woff2`
   - `Vazirmatn-Medium.woff2`
   - `Vazirmatn-Bold.woff2`

2. فایل‌های فونت را در مسیر زیر قرار دهید:
   ```
   safe-zone-map/
   └── assets/
       └── fonts/
           ├── Vazirmatn-Regular.woff2
           ├── Vazirmatn-Medium.woff2
           └── Vazirmatn-Bold.woff2
   ```

3. تعریف فونت در فایل `assets/css/style.css` از قبل انجام شده است.

> **نکته**: فونت Vazirmatn را می‌توانید از [GitHub](https://github.com/rastikerdar/vazirmatn) دریافت کنید.

## راهنمای استفاده 📖

### افزودن نقطه جدید
1. روی دکمه **"➕ افزودن نقطه جدید"** کلیک کنید
2. فرم را با اطلاعات مورد نظر پر کنید:
   - نام مکان (ضروری)
   - نوع (ضروری)
   - عرض و طول جغرافیایی (ضروری)
   - سایر اطلاعات (اختیاری)
3. روی **"💾 ذخیره"** کلیک کنید

### ویرایش نقطه
1. روی دکمه **"✏️ ویرایش"** در کارت مورد نظر کلیک کنید
2. تغییرات را اعمال کنید
3. روی **"💾 ذخیره"** کلیک کنید

### حذف نقطه
1. روی دکمه **"🗑️ حذف"** در کارت مورد نظر کلیک کنید
2. حذف را تأیید کنید

### جستجو و فیلتر
- **فیلتر بر اساس نوع**: از منوی کشویی بالای صفحه انتخاب کنید
- **جستجو**: در کادر جستجو تایپ کنید (در نام، آدرس و توضیحات جستجو می‌شود)

### خروجی گرفتن (Export)
1. روی دکمه **"📤 خروجی JSON"** کلیک کنید
2. فایل JSON به صورت خودکار دانلود می‌شود
3. این فایل شامل تمام نقاط ذخیره شده است

### وارد کردن داده‌ها (Import)
1. روی دکمه **"📥 ورودی JSON"** کلیک کنید
2. فایل JSON مورد نظر را انتخاب کنید
3. انتخاب کنید که داده‌ها جایگزین شوند یا به موارد موجود اضافه شوند

## امنیت و حریم خصوصی 🔒

- ✅ تمام داده‌ها فقط در مرورگر شما ذخیره می‌شوند
- ✅ هیچ اطلاعاتی به سرور ارسال نمی‌شود
- ✅ کاملاً آفلاین و بدون نیاز به اینترنت
- ⚠️ پاک کردن داده‌های مرورگر، اطلاعات را حذف می‌کند (حتماً از Export استفاده کنید)

## نکات مهم ⚠️

1. **پشتیبان‌گیری**: حتماً به صورت دوره‌ای از داده‌های خود خروجی JSON بگیرید
2. **پاک کردن cache**: پاک کردن cache مرورگر باعث حذف داده‌ها نمی‌شود، اما پاک کردن LocalStorage داده‌ها را حذف می‌کند
3. **محدودیت فضا**: LocalStorage محدودیت فضا دارد (معمولاً 5-10MB)
4. **تست قبل از استفاده**: حتماً قبل از استفاده واقعی، برنامه را تست کنید

## حل مشکلات رایج 🔍

### داده‌ها ذخیره نمی‌شوند
- مطمئن شوید که LocalStorage مرورگر فعال است
- حالت خصوصی (Incognito/Private) را غیرفعال کنید
- فضای کافی در LocalStorage وجود داشته باشد

### فونت درست نمایش داده نمی‌شود
- فایل‌های فونت را در مسیر `assets/fonts/` قرار دهید
- مطمئن شوید نام فایل‌ها دقیقاً مطابق است
- Console مرورگر را بررسی کنید (F12)

### فایل JSON وارد نمی‌شود
- مطمئن شوید فرمت JSON معتبر است
- فایل باید آرایه‌ای از اشیاء با ساختار صحیح باشد
