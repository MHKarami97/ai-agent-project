# برنامه وضعیت آب و هوا

یک برنامه ساده و زیبا برای نمایش وضعیت آب و هوای یک روز با استفاده از HTML، CSS و JavaScript.

## ویژگی‌ها

- رابط کاربری مدرن و زیبا
- نمایش دما، رطوبت و سرعت باد
- جستجوی شهر
- طراحی واکنش‌گرا (Responsive)
- پشتیبانی از زبان فارسی

## نحوه استفاده

1. **دریافت API Key:**
   - به سایت [OpenWeatherMap](https://openweathermap.org/api) بروید
   - یک حساب کاربری رایگان ایجاد کنید
   - API Key خود را دریافت کنید

2. **تنظیم API Key:**
   - فایل `script.js` را باز کنید
   - مقدار `YOUR_API_KEY_HERE` را با API Key خود جایگزین کنید

3. **اجرای برنامه:**
   - فایل `index.html` را در مرورگر باز کنید
   - نام شهر مورد نظر را وارد کرده و جستجو کنید

## فایل‌های پروژه

- `index.html` - ساختار HTML برنامه
- `style.css` - استایل‌های CSS
- `script.js` - منطق JavaScript و ارتباط با API

## نکات

- API Key رایگان OpenWeatherMap محدودیت دارد (60 درخواست در دقیقه)
- برای استفاده در محیط production، بهتر است API Key را در سمت سرور نگه دارید
