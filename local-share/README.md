# اشتراک‌گذاری فایل محلی (Local File Share)

یک برنامه وب ساده و امن برای اشتراک‌گذاری فایل‌ها بین دستگاه‌های موجود در یک شبکه محلی به صورت Peer-to-Peer (P2P).

## ویژگی‌ها

### 🌐 دو زبانه (فارسی/انگلیسی)
- پشتیبانی کامل از زبان فارسی و انگلیسی
- تغییر زبان به صورت آنی
- ذخیره‌سازی زبان انتخابی در مرورگر

### 🌓 حالت تاریک/روشن
- حالت روشن (Light Mode)
- حالت تاریک (Dark Mode)
- ذخیره‌سازی تم انتخابی در مرورگر

### 📁 اشتراک‌گذاری فایل
- ارسال فایل به صورت مستقیم بین دو دستگاه
- پشتیبانی از فایل‌های با هر حجمی
- نمایش پیشرفت انتقال فایل به صورت Real-time
- امکان ارسال چندین فایل به صورت همزمان
- Drag & Drop برای انتخاب فایل

### 🔒 امنیت
- اتصال P2P مستقیم بین دستگاه‌ها
- بدون نیاز به سرور مرکزی برای انتقال فایل
- داده‌ها مستقیماً بین دو دستگاه منتقل می‌شوند

### 💾 ذخیره‌سازی محلی
- ذخیره تنظیمات کاربر در LocalStorage مرورگر
- عدم نیاز به سرور Backend
- کاملاً سمت کاربر (Client-side)

## فناوری‌های استفاده شده

- **HTML5**: ساختار صفحه
- **CSS3**: طراحی و استایل‌دهی
- **JavaScript (ES6+)**: منطق برنامه
- **PeerJS**: ارتباط P2P بین دستگاه‌ها
- **LocalStorage**: ذخیره‌سازی تنظیمات کاربر
- **Vazirmatn Font**: فونت فارسی

## نحوه استفاده

### پیش‌نیازها

- یک مرورگر مدرن (Chrome, Firefox, Edge, Safari)
- اتصال به اینترنت (فقط برای برقراری اتصال اولیه)
- دستگاه‌ها باید در یک شبکه محلی یا قابل دسترسی به یکدیگر باشند

### راه‌اندازی

1. فایل `index.html` را در مرورگر باز کنید
2. یک شناسه یکتا (Peer ID) به صورت خودکار برای شما ایجاد می‌شود
3. شناسه خود را با کاربر مقصد به اشتراک بگذارید

### ارسال فایل

1. شناسه کاربر مقصد را در قسمت "شناسه دریافت‌کننده" وارد کنید
2. فایل مورد نظر را انتخاب کنید (کلیک یا Drag & Drop)
3. دکمه "ارسال فایل" را بزنید
4. کاربر مقصد درخواست شما را دریافت می‌کند
5. پس از تایید، فایل به صورت مستقیم منتقل می‌شود

### دریافت فایل

1. منتظر دریافت درخواست فایل باشید
2. با مشاهده اطلاعات فایل، آن را بپذیرید یا رد کنید
3. در صورت پذیرش، فایل به صورت خودکار دانلود می‌شود

## ساختار پروژه

```
local-share/
│
├── index.html              # صفحه اصلی برنامه
├── README.md              # مستندات پروژه
│
└── assets/
    ├── styles.css         # استایل‌های CSS
    ├── app.js            # منطق JavaScript
    └── translations.json  # فایل ترجمه‌ها
```

## تنظیمات PeerJS

برنامه از سرور عمومی PeerJS استفاده می‌کند:
- Host: `peerjs-server.herokuapp.com`
- Port: `443`
- Secure: `true`

برای استفاده در محیط خصوصی، می‌توانید سرور PeerJS خود را راه‌اندازی کنید:

```javascript
const PEERJS_CONFIG = {
    host: 'your-server.com',
    port: 9000,
    path: '/myapp',
    secure: true
};
```

## محدودیت‌ها

- برای برقراری اتصال اولیه نیاز به اینترنت است
- در صورت وجود Firewall یا NAT سخت‌گیرانه، ممکن است اتصال برقرار نشود
- حجم فایل محدودیتی ندارد اما سرعت به شبکه و دستگاه بستگی دارد

## رفع مشکلات رایج

### شناسه من نمایش داده نمی‌شود
- **چند ثانیه صبر کنید** - اتصال به سرور ممکن است 5-10 ثانیه طول بکشد
- **Console مرورگر را بررسی کنید** (F12) و به دنبال خطاها بگردید
- **اینترنت خود را چک کنید** - برای برقراری اتصال اولیه حتماً نیاز است
- **صفحه را Refresh کنید** - گاهی به سادگی این کار مشکل را حل می‌کند
- **مرورگر دیگری امتحان کنید** - Chrome یا Edge معمولاً بهتر کار می‌کنند
- **VPN یا Proxy خود را خاموش کنید** - ممکن است مانع اتصال شوند

### اتصال برقرار نمی‌شود
- مطمئن شوید هر دو دستگاه به اینترنت متصل هستند
- شناسه (Peer ID) را دوباره بررسی کنید
- صفحه را Refresh کنید
- Firewall یا آنتی‌ویروس را موقتاً غیرفعال کنید

### فایل ارسال نمی‌شود
- مطمئن شوید کاربر مقصد درخواست را پذیرفته است
- اتصال اینترنت خود را بررسی کنید
- از مرورگر دیگری امتحان کنید

### سرعت انتقال پایین است
- این به کیفیت شبکه محلی بستگی دارد
- در صورت امکان از اتصال سیمی استفاده کنید
- مطمئن شوید برنامه‌های دیگری پهنای باند را مصرف نمی‌کنند

## امنیت و حریم خصوصی

- همه فایل‌ها به صورت مستقیم بین دو دستگاه منتقل می‌شوند
- هیچ فایلی روی سرور ذخیره نمی‌شود
- فقط برای برقراری اتصال اولیه از سرور استفاده می‌شود
- توصیه می‌شود فقط با افراد مورد اعتماد فایل به اشتراک بگذارید

## توسعه و سفارشی‌سازی

### تغییر تم

رنگ‌های اصلی در فایل `assets/styles.css` قابل تغییر هستند:

```css
:root {
    --primary-color: #6366f1;
    --success-color: #10b981;
    --error-color: #ef4444;
    /* ... */
}
```

### اضافه کردن زبان جدید

در فایل `assets/translations.json`:

```json
{
  "fa": { ... },
  "en": { ... },
  "your-lang": {
    "app-title": "Your Translation",
    ...
  }
}
```

### تنظیمات اتصال

در فایل `assets/app.js`:

```javascript
const PEERJS_CONFIG = {
    host: 'your-custom-host',
    port: 9000,
    path: '/custom-path',
    secure: true
};
```
