# راهنمای استفاده سریع از برنامه‌ریز روزانه

## 🚀 شروع سریع

### گام 1: باز کردن برنامه
فایل `index.html` را در مرورگر خود باز کنید.

### گام 2: مشاهده تقویم
- تقویم شمسی در بالای صفحه نمایش داده می‌شود
- با دکمه‌های ◄ و ► بین ماه‌ها حرکت کنید
- روی هر روز کلیک کنید تا آن روز انتخاب شود

### گام 3: افزودن وظیفه
1. عنوان وظیفه را بنویسید (مثال: "جلسه کاری")
2. توضیحات اضافی (اختیاری): "جلسه با تیم فنی"
3. تاریخ: از تقویم انتخاب کنید
4. زمان: مثلاً 14:30
5. اولویت: زیاد/متوسط/کم
6. یادآوری: انتخاب کنید (مثلاً 15 دقیقه قبل)
7. دکمه "افزودن وظیفه" را بزنید

### گام 4: مدیریت وظایف
- ✅ برای تکمیل وظیفه، تیک کنار آن را بزنید
- ✏️ برای ویرایش، روی آیکون مداد کلیک کنید
- 🗑️ برای حذف، روی آیکون سطل زباله کلیک کنید

### گام 5: فیلتر و جستجو
- از دکمه‌های فیلتر (همه، امروز، این هفته، ...) استفاده کنید
- در کادر جستجو تایپ کنید تا وظایف فیلتر شوند

## 📊 ویژگی‌های کلیدی

### تقویم شمسی
- نمایش کامل تقویم جلالی
- نشان‌گذاری روزهایی که وظیفه دارند
- هایلایت روز امروز

### سطوح اولویت
- 🔵 **کم**: وظایف معمولی
- 🟡 **متوسط**: وظایف مهم
- 🔴 **زیاد**: وظایف فوری و مهم

### یادآوری‌ها
- بدون یادآوری
- در زمان وظیفه
- 5، 15، 30 دقیقه یا 1 ساعت قبل

### آمار
مشاهده آمار زنده:
- تعداد کل وظایف
- وظایف انجام شده
- وظایف در انتظار
- وظایف امروز

## 💡 نکات مفید

1. **ذخیره خودکار**: تمام تغییرات بلافاصله ذخیره می‌شوند
2. **یادآوری‌ها**: برای استفاده از یادآوری‌ها، دسترسی Notification را فعال کنید
3. **جستجو**: می‌توانید در عنوان و توضیحات وظایف جستجو کنید
4. **روز امروز**: روز فعلی با رنگ نارنجی نمایش داده می‌شود
5. **پاک کردن انجام شده‌ها**: با یک کلیک همه وظایف انجام شده را پاک کنید

## 🔐 امنیت داده‌ها

- تمام داده‌ها در LocalStorage مرورگر شما ذخیره می‌شوند
- هیچ اطلاعاتی به سرور ارسال نمی‌شود
- داده‌ها کاملاً محلی و امن هستند

## 🌐 استقرار روی GitHub Pages

```bash
# 1. ایجاد ریپازیتوری در GitHub
git init
git add .
git commit -m "Initial commit: Daily Planner"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/daily-planner.git
git push -u origin main

# 2. فعال‌سازی GitHub Pages
# به Settings > Pages بروید
# Source را روی main branch تنظیم کنید
# برنامه شما در https://YOUR_USERNAME.github.io/daily-planner/ در دسترس خواهد بود
```

## ❓ سوالات متداول

**س: آیا برنامه بدون اینترنت کار می‌کند؟**
ج: بله، پس از بارگذاری اولیه کاملاً آفلاین کار می‌کند.

**س: داده‌ها کجا ذخیره می‌شوند؟**
ج: در LocalStorage مرورگر شما، کاملاً محلی و امن.

**س: آیا می‌توانم از چند دستگاه استفاده کنم؟**
ج: هر مرورگر داده‌های جداگانه دارد. برای همگام‌سازی، باید داده‌ها را Export/Import کنید.

**س: یادآوری‌ها کار نمی‌کنند؟**
ج: دسترسی Notification را در تنظیمات مرورگر فعال کنید.

## 📱 تست در دستگاه‌های مختلف

برنامه روی تمام دستگاه‌ها به خوبی کار می‌کند:
- 💻 دسکتاپ
- 📱 موبایل
- 📲 تبلت

## 🎨 سفارشی‌سازی

برای تغییر رنگ‌ها، فایل `assets/css/style.css` را ویرایش کنید:

```css
:root {
    --primary-color: #6366f1;  /* رنگ اصلی */
    --success-color: #10b981;  /* رنگ موفقیت */
    --danger-color: #ef4444;   /* رنگ خطر */
}
```
