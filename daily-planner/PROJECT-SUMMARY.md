## 🚀 نحوه استفاده

### 1. استفاده محلی
```bash
# باز کردن فایل HTML در مرورگر
Start-Process "D:\Local\ai-agent-project\daily-planner\index.html"
```

### 2. استقرار روی GitHub Pages

```bash
# مقداردهی اولیه Git
cd D:\Local\ai-agent-project\daily-planner
git init
git add .
git commit -m "Initial commit: Daily Planner with Persian Calendar"

# اتصال به GitHub
git remote add origin https://github.com/YOUR_USERNAME/daily-planner.git
git branch -M main
git push -u origin main

# فعال‌سازی GitHub Pages
# Settings > Pages > Source: main branch
# URL: https://YOUR_USERNAME.github.io/daily-planner/
```

### 3. استفاده از سرور محلی
```bash
# با Python
python -m http.server 8000

# با Node.js
npx http-server

# با PHP
php -S localhost:8000
```
