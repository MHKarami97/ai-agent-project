# 📅 برنامه‌ریز روزانه (Daily Planner)

یک برنامه‌ریز روزانه کامل و کاربردی با پشتیبانی از تقویم شمسی (جلالی) برای مدیریت هوشمند وظایف روزانه

## ✨ ویژگی‌ها

- 📆 **تقویم شمسی**: نمایش کامل تقویم جلالی با امکان حرکت بین ماه‌ها
- ✅ **مدیریت وظایف**: افزودن، ویرایش، حذف و علامت‌گذاری وظایف
- 🔔 **یادآوری‌های هوشمند**: تنظیم یادآوری برای وظایف با زمان‌های مختلف
- 🎯 **سطوح اولویت**: تعیین اولویت کم، متوسط و زیاد برای وظایف
- 🔍 **جستجو و فیلتر**: جستجوی سریع و فیلتر وظایف بر اساس وضعیت
- 📊 **آمار و گزارش**: نمایش آمار کامل از وظایف
- 💾 **ذخیره‌سازی محلی**: استفاده از LocalStorage برای ذخیره دائمی داده‌ها
- 📱 **طراحی ریسپانسیو**: سازگار با تمام دستگاه‌ها
- 🎨 **رابط کاربری زیبا**: استفاده از فونت Vazirmatn و طراحی مدرن
- 🌙 **RTL کامل**: پشتیبانی کامل از راست‌چین فارسی

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها

این برنامه یک برنامه کاملاً Client-Side است و نیازی به سرور یا بکند ندارد. تنها به یک مرورگر مدرن نیاز دارید.

### نصب

1. **دانلود پروژه**
```bash
git clone https://github.com/YOUR_USERNAME/daily-planner.git
cd daily-planner
```

2. **اجرای برنامه**
   
   روش اول: باز کردن مستقیم فایل HTML
   - فایل `index.html` را در مرورگر خود باز کنید

   روش دوم: استفاده از یک سرور محلی ساده
   ```bash
   # با Python 3
   python -m http.server 8000
   
   # یا با Node.js
   npx http-server
   ```
   سپس به آدرس `http://localhost:8000` بروید

### استقرار روی GitHub Pages

1. فایل‌های پروژه را در ریپازیتوری GitHub خود قرار دهید
2. به تنظیمات ریپازیتوری بروید
3. از بخش **Pages** گزینه **Source** را روی `main branch` تنظیم کنید
4. برنامه شما در آدرس `https://YOUR_USERNAME.github.io/daily-planner/` در دسترس خواهد بود

## 🎯 نحوه استفاده

### افزودن وظیفه جدید

1. عنوان وظیفه را در فیلد "عنوان وظیفه" وارد کنید
2. در صورت نیاز توضیحات اضافی بنویسید
3. تاریخ را از تقویم انتخاب کنید
4. زمان وظیفه را مشخص کنید (اختیاری)
5. سطح اولویت را انتخاب کنید
6. در صورت نیاز یادآوری تنظیم کنید
7. دکمه "افزودن وظیفه" را بزنید

### ویرایش وظیفه

1. روی آیکون ویرایش (✏️) کنار وظیفه کلیک کنید
2. تغییرات مورد نظر را اعمال کنید
3. دکمه "ذخیره تغییرات" را بزنید

### حذف وظیفه

- روی آیکون حذف (🗑️) کنار وظیفه کلیک کنید
- حذف را تأیید کنید

### فیلتر کردن وظایف

از دکمه‌های فیلتر استفاده کنید:
- **همه**: نمایش تمام وظایف
- **امروز**: فقط وظایف امروز
- **این هفته**: وظایف هفته جاری
- **انجام شده**: وظایف تکمیل شده
- **در انتظار**: وظایف تکمیل نشده

### جستجو

در کادر جستجو متن مورد نظر را تایپ کنید تا وظایف مرتبط فیلتر شوند.

## 🔧 تکنولوژی‌های استفاده شده

- **HTML5**: ساختار صفحه
- **CSS3**: استایل‌دهی و طراحی ریسپانسیو
- **JavaScript (Vanilla)**: منطق برنامه بدون استفاده از فریمورک
- **LocalStorage API**: ذخیره‌سازی داده‌ها در مرورگر
- **Notification API**: نمایش یادآوری‌ها
- **Vazirmatn Font**: فونت فارسی زیبا

## ⚙️ ویژگی‌های تکنیکی

### تبدیل تاریخ

برنامه شامل توابع کامل تبدیل تاریخ میلادی به شمسی و بالعکس است:
- `gregorianToPersian(year, month, day)`: تبدیل میلادی به شمسی
- `persianToGregorian(year, month, day)`: تبدیل شمسی به میلادی
- پشتیبانی از سال‌های کبیسه

### ذخیره‌سازی داده

- تمام داده‌ها در LocalStorage مرورگر ذخیره می‌شوند
- داده‌ها حتی پس از بستن مرورگر باقی می‌مانند
- امکان پشتیبان‌گیری و بازیابی داده‌ها

### یادآوری‌ها

- استفاده از Notification API برای نمایش یادآوری‌ها
- بررسی خودکار یادآوری‌ها هر دقیقه
- یادآوری‌های زمان‌دار: 0، 5، 15، 30 و 60 دقیقه قبل

## 🎨 سفارشی‌سازی

### تغییر رنگ‌ها

رنگ‌های برنامه در فایل `assets/css/style.css` در بخش `:root` تعریف شده‌اند:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --success-color: #10b981;
    --danger-color: #ef4444;
    /* ... */
}
```

### تغییر فونت

برای استفاده از فونت دیگر، فایل CSS را ویرایش کنید:

```css
@import url('YOUR_FONT_URL');

body {
    font-family: 'YOUR_FONT_NAME', sans-serif;
}
```

## 🐛 رفع مشکلات رایج

### یادآوری‌ها کار نمی‌کنند

- اطمینان حاصل کنید که دسترسی Notification را برای سایت فعال کرده‌اید
- در تنظیمات مرورگر، دسترسی‌های سایت را بررسی کنید

### داده‌ها ذخیره نمی‌شوند

- بررسی کنید که LocalStorage در مرورگر شما فعال باشد
- حالت Incognito/Private ممکن است LocalStorage را غیرفعال کند

### تقویم درست نمایش داده نمی‌شود

- کش مرورگر را پاک کنید
- صفحه را رفرش کنید (Ctrl+F5 یا Cmd+Shift+R)
