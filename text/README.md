# ژنراتور Lorem Ipsum فارسی

این پروژه یک صفحهٔ ثابت HTML/CSS/JS است که در مرورگر اجرا می‌شود و بدون نیاز به سرور خارجی، متن تستی فارسی تولید می‌کند. قالب پاسخگوست و برای نمایش در نمایشگرهای مختلف، از جمله موبایل، بهینه‌سازی شده است.

## ساختار پروژه

- `index.html` – ساختار HTML صفحه شامل هدر، فرم تنظیمات، و بخش نمایش متن.
- `assets/css/styles.css` – سبک‌دهی پاسخگو با دستور زبان راست‌به‌چپ.
- `assets/js/app.js` – منطق تولید متن، به‌روزرسانی نمای، و ذخیره‌سازی ترجیحات کاربر در مرورگر.

## نحوه استفاده

1. فولدر پروژه را باز کنید و فایل `index.html` را در مرورگر دلخواه باز کنید.
2. با تنظیم تعداد پاراگراف و تعداد جملات در هر پاراگراف، روی دکمه «تولید جدید» کلیک کنید تا متن تازه نمایش داده شود.
3. اگر نیاز به اشتراک‌گذاری متن دارید، دکمه «کپی متن» فعال می‌شود و می‌توانید متن را کپی کنید.

## ویژگی‌ها

- رابط کاربری پاسخگو برای نمایش روی نمایشگرهای کوچک و بزرگ.
- پشتیبانی از نمایش راست‌به‌چپ و فونت فارسی برای حفظ انسجام زبان.
- ذخیره تنظیمات جاری در مرورگر تا در بازدیدهای بعدی موثر باشد.

## توسعه و آزمایش

- هر تغییری در `assets/css/styles.css` یا `assets/js/app.js` را ذخیره کنید و مرورگر را مجدداً بارگذاری کنید تا تاثیر آن را ببینید.
- برای تست روی سرویس‌های ساده می‌توانید از `npx http-server` یا هر ابزار مشابه برای سرو کردن فولدر استفاده کنید.

## نکات

- حتی بدون سرور، می‌توانید پروژه را روی چنین سرویس‌هایی هاست کنید و از امکانات مرورگر برای ذخیره‌سازی استفاده کنید.
