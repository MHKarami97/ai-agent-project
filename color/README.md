# ژنراتور پالت رنگ هماهنگ

یک صفحه‌ی ثابت RTL که در مرورگر کار می‌کند و رنگ‌های هماهنگ را از رنگ پایه می‌سازد، پالت‌ها را ذخیره می‌کند و کد CSS آن‌ها را کپی می‌کند.

## ساختار پروژه

- `index.html` – صفحه اصلی با لینک به CSS/JS در پوشه‌ی `assets`، فونت Vazirmatn و رابط کاربری RTL.
- `assets/css/styles.css` – استایل‌های واکنش‌گرا، تنظیمات تایپوگرافی و کارت‌های پالت. فونت Vazirmatn از CDN بارگذاری می‌شود و روی عناصر ورودی/دکمه نیز اعمال شده.
- `assets/js/app.js` – منطق تولید رنگ، ذخیره با `localStorage`، بارگذاری پالت‌های ذخیره‌شده، نوتیفیکیشن و کپی CSS.
- `assets/fonts/` – (اختیاری) اگر نیاز داشتید می‌توانید فایل‌های Vazirmatn را این‌جا بگذارید و در `styles.css` `@font-face` را به آن‌ها اشاره دهید.

## ویژگی‌ها

1. RTL کامل و استفاده از فونت Vazirmatn برای تمامی عناصر شامل `button`، `input` و کل صفحه.
2. تولید پالت بین ۳ تا ۶ رنگ با تنظیمات پویای کشویی و انتخاب رنگ پایه.
3. ذخیره‌ی حداکثر ۱۲ پالت در `localStorage` مرورگر، قابل بارگذاری دوباره.
4. امکان کپی کد CSS پالت برای استفاده در پروژه‌های دیگر.
5. نوتیفیکیشن موقت برای اطلاع‌رسانی هنگام ذخیره، بارگذاری و کپی.

## نحوه اجرا

۱. در محیط توسعه، فایل `index.html` را با مرورگر باز کنید (بدون نیاز به سرور):

```powershell
cd D:\Local\ai-agent-project\color
start index.html
```

۲. تنظیمات رنگ را تغییر دهید، پالت را ذخیره یا بارگذاری کنید.
۳. برای انتشار روی GitHub Pages:

```powershell
git init          # اگر هنوز گیت ندارد
git add .
git commit -m "Deploy palette generator"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

سپس در تنظیمات ریپازیتوری، GitHub Pages را روی شاخه `main` با `docs/` یا `root` تنظیم کنید (برای این پروژه، روت است).

## نکات اضافی

- همه چیز سمت مرورگر انجام می‌شود؛ برای ذخیره‌ی پالت از `localStorage` استفاده می‌شود تا هنگام باز/بستن مرورگر، پالت‌ها محفوظ بمانند.
- به دلیل RTL بودن، layout برای زبان‌های راست به چپ بهینه شده و عناصر ورودی مشابه فضای فارسی طراحی شدند.
- اگر خواستید فونت را لوکال نگه دارید، فایل‌های Vazirmatn را در `assets/fonts` بگذارید و سورس `@font-face` را به آن‌ها اشاره دهید.
