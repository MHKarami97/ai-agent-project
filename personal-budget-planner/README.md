# 💰 برنامه مدیریت بودجه شخصی

یک برنامه وب کامل و حرفه‌ای برای مدیریت و برنامه‌ریزی بودجه شخصی با قابلیت تحلیل و نمودارهای تعاملی.

## ✨ ویژگی‌ها

- 📊 **نمودارهای تحلیلی پیشرفته**: نمودار دایره‌ای، میله‌ای و خطی برای تحلیل دقیق درآمد و هزینه‌ها
- 💾 **ذخیره‌سازی محلی**: استفاده از IndexedDB برای ذخیره امن داده‌ها در مرورگر
- ✏️ **مدیریت کامل تراکنش‌ها**: امکان افزودن، ویرایش و حذف تراکنش‌ها
- 🔍 **جستجو و فیلتر**: جستجوی سریع و فیلتر بر اساس نوع تراکنش
- 📈 **خلاصه مالی**: نمایش لحظه‌ای کل درآمد، هزینه و موجودی
- 📱 **طراحی واکنش‌گرا**: سازگار با تمام دستگاه‌ها (موبایل، تبلت، دسکتاپ)
- 🎨 **رابط کاربری زیبا**: طراحی مدرن و جذاب با انیمیشن‌های نرم
- 🔤 **فونت Vazirmatn**: استفاده از فونت فارسی زیبا و خوانا در تمام المان‌ها
- 📥 **خروجی CSV**: امکان دانلود تمام تراکنش‌ها در قالب فایل Excel
- 🌙 **RTL کامل**: پشتیبانی کامل از راست به چپ برای زبان فارسی

## 🚀 نحوه استفاده

### استفاده محلی

1. فایل‌ها را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید
3. شروع به استفاده کنید!

### استقرار در GitHub Pages

1. مخزن را Fork کنید یا فایل‌ها را در مخزن خود آپلود کنید
2. به تنظیمات مخزن بروید (Settings)
3. به بخش Pages بروید
4. در قسمت Source گزینه `main` branch را انتخاب کنید
5. روی Save کلیک کنید
6. بعد از چند دقیقه، لینک سایت شما آماده می‌شود

## 🛠️ تکنولوژی‌های استفاده شده

- **HTML5**: ساختار صفحه
- **CSS3**: طراحی و استایل‌دهی با انیمیشن‌های پیشرفته
- **JavaScript (ES6+)**: منطق برنامه و تعاملات
- **IndexedDB**: دیتابیس مرورگر برای ذخیره داده‌ها
- **Chart.js**: کتابخانه نمودارهای تعاملی
- **Vazirmatn Font**: فونت فارسی زیبا و مدرن

## 📊 قابلیت‌های نمودارها

### نمودار دایره‌ای (Pie Chart)
نمایش درصد هزینه‌ها بر اساس دسته‌بندی

### نمودار میله‌ای (Bar Chart)
مقایسه درآمد و هزینه در هر دسته‌بندی

### نمودار خطی (Line Chart)
نمایش روند درآمد و هزینه در طول زمان (ماهانه)

## 💡 نحوه استفاده از ویژگی‌ها

### افزودن تراکنش
1. نوع تراکنش (درآمد یا هزینه) را انتخاب کنید
2. دسته‌بندی مناسب را انتخاب کنید
3. توضیحات تراکنش را وارد کنید
4. مبلغ را به تومان وارد کنید
5. تاریخ را انتخاب کنید
6. روی "افزودن تراکنش" کلیک کنید

### ویرایش تراکنش
1. روی دکمه "ویرایش" تراکنش مورد نظر کلیک کنید
2. اطلاعات را ویرایش کنید
3. روی "ذخیره تغییرات" کلیک کنید

### حذف تراکنش
1. روی دکمه "حذف" تراکنش مورد نظر کلیک کنید
2. تایید حذف را انجام دهید

### جستجو و فیلتر
- از کادر جستجو برای یافتن تراکنش‌های خاص استفاده کنید
- از دکمه‌های "همه"، "درآمد" و "هزینه" برای فیلتر کردن استفاده کنید

### خروجی Excel
- روی دکمه "خروجی Excel" کلیک کنید
- فایل CSV شامل تمام تراکنش‌ها دانلود می‌شود
- فایل را با Excel یا هر نرم‌افزار مشابه باز کنید

## 🐛 رفع مشکلات

### داده‌ها نمایش داده نمی‌شوند
- مطمئن شوید که JavaScript فعال است
- Cache مرورگر را پاک کنید
- صفحه را رفرش کنید

### نمودارها نمایش داده نمی‌شوند
- مطمئن شوید که حداقل یک تراکنش ثبت شده است
- اتصال اینترنت خود را بررسی کنید (برای بارگذاری Chart.js)

### فونت فارسی نمایش داده نمی‌شود
- مطمئن شوید که اتصال اینترنت برقرار است
- Cache مرورگر را پاک کنید
