# تبدیل‌کننده تقویم و محاسبه‌گر سن

ابزار تحت مرورگر برای تبدیل تاریخ بین تقویم‌های **شمسی، میلادی و قمری** و همچنین محاسبه سن دقیق افراد. همه چیز در سمت کاربر اجرا می‌شود بنابراین می‌توان آن را به‌راحتی روی GitHub Pages منتشر کرد.

## ویژگی‌ها
- رابط کاربری راست‌به‌چپ با فونت Vazirmatn برای تمام اجزا
- تبدیل تاریخ بین سه تقویم رایج
- محاسبه سن بر اساس تاریخ تولد در هر تقویم
- ذخیره تاریخچه تبدیل‌ها و نتایج سن‌سنجی با `localStorage`
- بدون وابستگی به سرور؛ مناسب برای GitHub Pages

## ساختار پروژه
```
.
├── index.html
├── assets
│   ├── css
│   │   └── style.css
│   ├── js
│   │   ├── app.js
│   │   └── converters.js
│   └── fonts
│       └── Vazirmatn-Regular.woff2
└── README.md
```

## شروع سریع
1. مخزن را کلون یا دانلود کنید.
2. فایل `index.html` را در مرورگر باز کنید.
3. از فرم‌ها برای تبدیل تاریخ یا محاسبه سن استفاده کنید. نتیجه‌ها در همان صفحه نمایش داده می‌شوند.

## استقرار روی GitHub Pages
1. پروژه را به مخزن جدید در GitHub پوش کنید.
2. در تنظیمات مخزن، از بخش **Pages** شاخه `main` و ریشه (`/`) را انتخاب کنید.
3. چند دقیقه بعد آدرس GitHub Pages فعال می‌شود و سایت در دسترس خواهد بود.

## توسعه
- کدهای استایل در `assets/css/style.css` قرار دارد.
- منطق برنامه در `assets/js/app.js` و توابع تبدیل در `assets/js/converters.js` هستند.
- برای اضافه کردن فونت یا فایل دیگر از مسیر `assets/` استفاده کنید.

## تست محلی
فقط به یک مرورگر مدرن نیاز دارید. اگر می‌خواهید با سرور محلی آزمایش کنید:
```bash
python -m http.server
```
سپس مرورگر را روی `http://localhost:8000` باز کنید.
