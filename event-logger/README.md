# مستندساز رویداد (Event Logger)

یک برنامه تحت وب برای ثبت و مدیریت رویدادها با قابلیت ذخیره‌سازی محلی در مرورگر

## 📋 ویژگی‌ها

### امکانات اصلی
- ✅ **ثبت رویداد با جزئیات کامل**: عنوان، تاریخ و زمان، توضیحات
- 📷 **ضمیمه تصویر**: امکان اضافه کردن تصویر به هر رویداد
- 📍 **موقعیت مکانی**: ثبت موقعیت جغرافیایی (GPS) و آدرس
- 🔍 **جستجو و فیلتر**: جستجوی سریع در رویدادها
- 🔄 **مرتب‌سازی**: مرتب‌سازی بر اساس تاریخ یا عنوان
- ✏️ **ویرایش و حذف**: امکان ویرایش و حذف رویدادهای ثبت شده
- 👁️ **نمایش جزئیات**: مشاهده کامل اطلاعات رویداد در مودال
- 📦 **خروجی رمزشده**: تهیه نسخه پشتیبان رمزنگاری شده
- 📥 **وارد کردن**: بازیابی داده‌ها از فایل پشتیبان
- 💾 **ذخیره‌سازی محلی**: تمام داده‌ها در IndexedDB مرورگر ذخیره می‌شوند

### ویژگی‌های فنی
- 🚀 بدون نیاز به سرور بک‌اند
- 🔒 رمزنگاری داده‌ها با رمز عبور دلخواه
- 📱 طراحی کاملاً ریسپانسیو
- 🎨 رابط کاربری زیبا و مدرن
- ⚡ سرعت بالا و کارایی عالی
- 🌐 کاملاً به زبان فارسی با فونت Vazirmatn

## 🚀 نحوه استفاده

### نصب و راه‌اندازی

1. **دانلود و استخراج فایل‌ها**
   - تمام فایل‌های پروژه را در یک پوشه قرار دهید

2. **تنظیم فونت Vazirmatn**
   - فایل‌های فونت Vazirmatn را در پوشه `assets/fonts/` قرار دهید
   - فایل‌های مورد نیاز:
     - `Vazirmatn-Regular.woff2`
     - `Vazirmatn-Medium.woff2`
     - `Vazirmatn-Bold.woff2`

3. **اجرای برنامه**
   - فایل `index.html` را در مرورگر خود باز کنید
   - یا از یک وب‌سرور محلی استفاده کنید:
     ```bash
     # با Python
     python -m http.server 8000
     
     # با Node.js (http-server)
     npx http-server
     
     # با PHP
     php -S localhost:8000
     ```
   - سپس به آدرس `http://localhost:8000` بروید

## 📝 راهنمای استفاده

### افزودن رویداد جدید

1. در بخش "افزودن رویداد جدید"، فرم را پر کنید:
   - **عنوان رویداد**: (الزامی) عنوان رویداد را وارد کنید
   - **تاریخ و زمان**: (الزامی) تاریخ و زمان رویداد را انتخاب کنید
   - **توضیحات**: (اختیاری) جزئیات بیشتر درباره رویداد
   - **تصویر**: (اختیاری) یک تصویر برای رویداد انتخاب کنید
   - **موقعیت مکانی**: (اختیاری) آدرس یا نام مکان را وارد کنید
     - می‌توانید با کلیک روی "دریافت موقعیت فعلی" موقعیت GPS خود را ثبت کنید

2. روی دکمه "✅ ذخیره رویداد" کلیک کنید

### مشاهده و مدیریت رویدادها

- **مشاهده جزئیات**: روی آیکون 👁️ کلیک کنید
- **ویرایش**: روی آیکون ✏️ کلیک کنید
- **حذف**: روی آیکون 🗑️ کلیک کنید

### جستجو و فیلتر

- در کادر جستجو، عبارت مورد نظر را تایپ کنید
- رویدادها به صورت لحظه‌ای فیلتر می‌شوند
- جستجو در عنوان، توضیحات و موقعیت مکانی انجام می‌شود

### مرتب‌سازی

از منوی کشویی مرتب‌سازی یکی از گزینه‌های زیر را انتخاب کنید:
- جدیدترین
- قدیمی‌ترین
- براساس عنوان (الف - ی)
- براساس عنوان (ی - الف)

### تهیه نسخه پشتیبان (خروجی رمزشده)

1. روی دکمه "📦 خروجی رمزشده" کلیک کنید
2. رمز عبور دلخواه خود را وارد کنید
3. فایل JSON رمزنگاری شده دانلود می‌شود

⚠️ **مهم**: رمز عبور را به خاطر بسپارید! بدون آن نمی‌توانید داده‌ها را بازیابی کنید

### بازیابی از نسخه پشتیبان

1. روی دکمه "📥 وارد کردن" کلیک کنید
2. فایل پشتیبان را انتخاب کنید
3. رمز عبوری که هنگام ایجاد پشتیبان وارد کردید را تایپ کنید
4. تأیید کنید که می‌خواهید داده‌های فعلی جایگزین شوند

## 🔧 تکنولوژی‌ها

- **HTML5**: ساختار صفحه
- **CSS3**: استایل‌دهی و انیمیشن‌ها
- **JavaScript (ES6+)**: منطق برنامه
- **IndexedDB**: ذخیره‌سازی داده‌ها در مرورگر
- **Geolocation API**: دریافت موقعیت GPS
- **FileReader API**: خواندن و نمایش تصاویر

## 🌟 ویژگی‌های امنیتی

- 🔐 رمزنگاری XOR با Base64 برای خروجی
- 💾 ذخیره‌سازی کاملاً محلی (هیچ داده‌ای به سرور ارسال نمی‌شود)
- 🔒 رمز عبور برای خروجی و وارد کردن داده‌ها
- 🛡️ Escape کردن محتوای HTML برای جلوگیری از XSS

## 📱 سازگاری مرورگرها

این برنامه با تمام مرورگرهای مدرن سازگار است:
- ✅ Chrome / Edge (نسخه 58+)
- ✅ Firefox (نسخه 55+)
- ✅ Safari (نسخه 11+)
- ✅ Opera (نسخه 45+)

## ⚠️ نکات مهم

1. **ذخیره‌سازی محلی**: تمام داده‌ها فقط در مرورگر شما ذخیره می‌شوند. در صورت پاک کردن داده‌های مرورگر، رویدادها حذف خواهند شد.

2. **نسخه پشتیبان منظم**: حتماً به صورت منظم از رویدادهای خود نسخه پشتیبان تهیه کنید.

3. **رمز عبور قوی**: برای خروجی رمزشده از رمز عبور قوی استفاده کنید.

4. **موقعیت مکانی**: برای استفاده از GPS، باید به مرورگر اجازه دسترسی به موقعیت مکانی خود را بدهید.

5. **اندازه تصاویر**: تصاویر بزرگ ممکن است فضای زیادی اشغال کنند. سعی کنید از تصاویر با حجم مناسب استفاده کنید.

## 🎨 سفارشی‌سازی

### تغییر رنگ‌ها

در فایل `assets/css/style.css` متغیرهای رنگی را تغییر دهید:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --success-color: #10b981;
    --danger-color: #ef4444;
    /* ... */
}
```

### تغییر فونت

اگر می‌خواهید از فونت دیگری استفاده کنید، تنظیمات `@font-face` در ابتدای فایل CSS را تغییر دهید.

## 🐛 رفع مشکلات

### رویدادها نمایش داده نمی‌شوند
- کنسول مرورگر را باز کنید (F12) و خطاها را بررسی کنید
- مطمئن شوید JavaScript فعال است
- Cache مرورگر را پاک کنید

### موقعیت GPS کار نمی‌کند
- به مرورگر اجازه دسترسی به موقعیت مکانی را بدهید
- مطمئن شوید اتصال اینترنت فعال است (برای برخی مرورگرها)
- از HTTPS یا localhost استفاده کنید

### تصاویر نمایش داده نمی‌شوند
- مطمئن شوید فایل انتخابی یک تصویر معتبر است
- اندازه فایل را کاهش دهید
- فرمت‌های پشتیبانی شده: JPG, PNG, GIF, WebP

### خطا در وارد کردن داده‌ها
- مطمئن شوید رمز عبور صحیح است
- فایل انتخابی باید یک خروجی معتبر از این برنامه باشد
- فایل نباید دستکاری شده باشد
