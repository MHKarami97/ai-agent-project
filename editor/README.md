# ویرایشگر مارک‌داون RTL

ویرایشگر کاملاً سمت مرورگر با تمرکز بر تجربه کاربری راست‌به‌چپ برای کاربران فارسی‌زبان. این پروژه روی GitHub Pages به‌راحتی قابل استقرار است و امکانات پیشرفته‌ای مثل پیش‌نمایش زنده، ذخیره خودکار، خروجی PDF/Word و تم روشن/تیره دارد.

## امکانات کلیدی
- **ساختار تفکیک‌شده**: `index.html` + پوشه‌های `assets/css` و `assets/js`.
- **واکنش‌گرا و RTL**: طراحی فلکس/گرید، سازگار با دسکتاپ و موبایل.
- **فونت Vazirmatn**: برای تمام متون، ورودی‌ها و دکمه‌ها.
- **Markdown پیشرفته**: با کتابخانه `marked` و پاک‌سازی `DOMPurify`.
- **پیش‌نمایش زنده**: هم‌زمان با تایپ.
- **ذخیره خودکار**: استفاده از LocalStorage با پیام وضعیت.
- **خروجی فایل**: PDF با `html2pdf.js` و Word با `html-docx-js`.
- **تم دوگانه**: سوئیچ سریع بین حالت روشن/تیره.

## ساختار پروژه
```
index.html
assets/
  css/
    styles.css
  js/
    app.js
README.md
```

## اجرای محلی
1. مخزن را کلون کنید یا فایل‌ها را دانلود نمایید.
2. فایل `index.html` را در مرورگر باز کنید (نیازی به سرور نیست).

> برای توسعه، می‌توانید از افزونه‌هایی مثل Live Server استفاده کنید تا رفرش خودکار داشته باشید.

## استقرار روی GitHub Pages
1. مخزن را در GitHub بسازید و این فایل‌ها را پوش کنید.
2. به تنظیمات مخزن بروید و در بخش **Pages**، شاخه اصلی (مثلاً `main`) و ریشه (`/root`) را انتخاب کنید.
3. پس از چند دقیقه، لینک عمومی سایت فعال می‌شود.

## نحوه استفاده
- متن مارک‌داون خود را در پنل سمت راست وارد کنید.
- تغییرات به‌صورت زنده در پنل پیش‌نمایش نمایش داده می‌شود.
- با کلیدهای `Ctrl/Cmd + S` ذخیره دستی بگیرید؛ سیستم ذخیره خودکار نیز فعال است.
- برای شروع سریع از دکمه «نمونهٔ مارک‌داون» استفاده کنید.
- خروجی PDF یا Word تنها با یک کلیک قابل دانلود است.

## وابستگی‌ها
تمام وابستگی‌ها از CDN بارگذاری می‌شوند:
- [`marked`](https://github.com/markedjs/marked)
- [`dompurify`](https://github.com/cure53/DOMPurify)
- [`html2pdf.js`](https://github.com/eKoopmans/html2pdf.js)
- [`html-docx-js`](https://github.com/evidenceprime/html-docx-js)

## نکات توسعه
- برای تغییر تم رنگی، مقادیر CSS داخل `:root` را ویرایش کنید.
- اگر می‌خواهید فونت‌ها را آفلاین داشته باشید، نسخه‌های `.woff2` را دانلود کرده و مسیرها را در `@font-face` تغییر دهید.
- منطق مربوط به ذخیره، تم و خروجی در `assets/js/app.js` قابل ارتقا است.
