# Base64 Encoder/Decoder

این پروژه یک ابزار ساده برای تبدیل متن‌ها و تصویرها به Base64 و برعکس روی مرورگر است.

## ساختار پروژه

- `index.html` : رابط کاربری با جهت راست به چپ و اتصال به فایل‌های مربوطه.
- `assets/css/styles.css` : حالت‌های واکنشگرا، رنگ‌بندی و فاصله‌دهی فرم‌ها، دکمه‌ها و کارت‌ها.
- `assets/js/encoder.js` : منطق تبدیل متن، مدیریت پیش‌نمایش تصاویر و دکمه‌های کپی.

## نحوه استفاده

1. یکی از مرورگرهای مدرن را باز کرده و `index.html` را اجرا کنید (با دوبار کلیک یا از طریق سرویس‌دهنده محلی).
2. برای تبدیل متن، آن را در کادر مخصوص وارد کنید و روی «رمزنگاری» یا «بازگشایی» کلیک کنید.
3. تصویر را انتخاب کنید تا رشته Base64 تولید شده در خروجی نمایش داده شود؛ سپس می‌توانید با دکمه‌های موجود آن را کپی یا پاک کنید.
4. در صورت نیاز می‌توانید متن یا تصویر جدیدی را وارد کرده و مجدداً عمل تبدیل یا بازسازی را اجرا کنید.

## نکات

- طراحی واکنشگرا باعث می‌شود که فرم‌ها و کارت‌ها به‌خوبی روی موبایل نمایش داده شوند.
- پیش‌نمایش تصویر پس از انتخاب یا وارد کردن رشته Base64 به‌صورت خودکار به‌روز می‌شود.
