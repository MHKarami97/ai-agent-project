# مبدل فرمت تصویر در مرورگر

وب‌اپلیکیشن کاملاً سمت کاربر برای تبدیل فرمت‌های رایج تصویر (PNG, JPG, JPEG, GIF, WEBP) بدون نیاز به سرور. رابط کاربری فارسی، راست‌به‌چپ و مبتنی بر فونت [Vazirmatn](https://github.com/rastikerdar/vazirmatn).

## ویژگی‌ها
- اجرای کامل در مرورگر با استفاده از FileReader و Canvas
- پشتیبانی از کیفیت قابل تنظیم برای JPG/WEBP
- تاریخچه تبدیل ذخیره‌شده در `localStorage` قابل پاک‌سازی
- طراحی واکنش‌گرا و سازگار با موبایل
- آماده استقرار روی GitHub Pages بدون نیاز به بیلد

## ساختار پروژه
```
.
├── index.html
├── assets
│   ├── css
│   │   └── style.css
│   ├── js
│   │   └── app.js
│   └── fonts/ (اختیاری برای میزبانی فونت)
└── README.md
```

## شروع به کار محلی
1. مخزن را کلون یا دانلود کنید.
2. فایل `index.html` را مستقیماً در مرورگر باز کنید یا یک سرور محلی ساده اجرا کنید:

```bash
cd convertor
python -m http.server 8080
```

3. مرورگر را به آدرس `http://localhost:8080` هدایت کنید.

## نحوه استفاده
1. تصویر را از دستگاه خود انتخاب کنید.
2. فرمت خروجی را (PNG/JPG/WEBP/GIF) برگزینید و در صورت نیاز کیفیت را تنظیم کنید.
3. روی «تبدیل کن» بزنید؛ پیش‌نمایش و لینک دانلود ایجاد می‌شود.
4. تاریخچه تبدیل‌های اخیر در پایین صفحه ذخیره می‌شود و می‌توانید آن را پاک کنید.

## استقرار روی GitHub Pages
1. سورس را در شاخه اصلی (مثلاً `main`) پوش کنید.
2. در تنظیمات مخزن به **Pages** بروید و Branch را روی همان شاخه و مسیر `/ (root)` قرار دهید.
3. پس از چند دقیقه سایت با آدرس `https://USERNAME.github.io/REPO` در دسترس است.

## تست و نکات
- تبدیل GIF تنها یک فریم ثابت می‌سازد (انیمیشن پشتیبانی نمی‌شود).
- کیفیت برای PNG/GIF نادیده گرفته می‌شود (فرمت‌های lossless).
- برای حفظ عملکرد، اندازه تاریخچه به ۸ مورد محدود شده است.
- برای پاک‌سازی تاریخچه آفلاین می‌توانید کلید «پاک‌سازی» یا ابزارهای مرورگر را برای حذف `localStorage` استفاده کنید.
