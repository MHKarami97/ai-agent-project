# تابلوی اعلانات محلی (Local Board)

<div dir="rtl">

## 📋 درباره پروژه

تابلوی اعلانات محلی یک وب‌اپلیکیشن کاملاً فرانت‌اند است که به کاربران امکان می‌دهد پیام‌ها و اطلاعات خود را با همسایگان و افراد در محدوده نزدیک به اشتراک بگذارند. این پروژه بدون نیاز به سرور بکند کار می‌کند و تمام داده‌ها در localStorage مرورگر ذخیره می‌شوند.

### ✨ ویژگی‌های اصلی

- **💬 ارسال و مشاهده پیام‌ها**: امکان ایجاد، مشاهده و حذف پیام‌های مختلف
- **📂 دسته‌بندی پیام‌ها**: تقسیم‌بندی پیام‌ها به دسته‌های مختلف (عمومی، اخبار، خرید و فروش، خدمات، رویداد، اضطراری)
- **🔍 فیلتر کردن پیام‌ها**: جستجوی آسان پیام‌ها بر اساس دسته‌بندی
- **🔔 سیستم اعلان‌ها**: دریافت نوتیفیکیشن برای پیام‌های جدید
- **💾 ذخیره‌سازی محلی**: ذخیره تمام پیام‌ها در مرورگر بدون نیاز به سرور
- **🎨 رابط کاربری زیبا**: طراحی مدرن و کاربرپسند با فونت Vazirmatn
- **📱 ریسپانسیو**: سازگار با تمام ابعاد صفحه نمایش (موبایل، تبلت، دسکتاپ)
- **🌙 انیمیشن‌های روان**: تجربه کاربری دلنشین با انیمیشن‌های نرم
- **♿ قابلیت دسترسی**: پشتیبانی از استانداردهای WCAG
- **⚡ عملکرد بالا**: بارگذاری سریع و بدون نیاز به اینترنت

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها

این پروژه فقط به یک مرورگر وب مدرن نیاز دارد. هیچ ابزار یا نرم‌افزار دیگری لازم نیست.

### مراحل نصب

1. **دانلود پروژه**
   ```bash
   # دانلود یا کلون کردن پروژه
   git clone <repository-url>
   cd local-board
   ```

2. **اضافه کردن فایل‌های فونت**
   
   فایل‌های فونت Vazirmatn را در مسیر زیر قرار دهید:
   ```
   local-board/
   └── assets/
       └── fonts/
           ├── Vazirmatn-Regular.woff2
           ├── Vazirmatn-Regular.woff
           ├── Vazirmatn-Medium.woff2
           ├── Vazirmatn-Medium.woff
           ├── Vazirmatn-Bold.woff2
           └── Vazirmatn-Bold.woff
   ```

   **نکته**: فایل‌های فونت را می‌توانید از [وب‌سایت رسمی Vazirmatn](https://github.com/rastikerdar/vazirmatn) دریافت کنید.

3. **اجرای پروژه**
   
   فایل `index.html` را در مرورگر خود باز کنید:
   - دوبار کلیک روی فایل `index.html`
   - یا کلیک راست و انتخاب "Open with" و انتخاب مرورگر
   - یا استفاده از یک سرور محلی:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js (با نصب http-server)
     npx http-server
     
     # PHP
     php -S localhost:8000
     ```
   
   سپس به آدرس `http://localhost:8000` بروید.

## 🎯 نحوه استفاده

### 1. ارسال پیام جدید

1. در بخش "پیام جدید" عنوان پیام را وارد کنید
2. متن پیام خود را در قسمت "متن پیام" بنویسید
3. دسته‌بندی مناسب را انتخاب کنید
4. نام خود را وارد کنید (اختیاری)
5. روی دکمه "ارسال پیام" کلیک کنید

**میانبر صفحه کلید**: `Ctrl + Enter` برای ارسال سریع پیام

### 2. مشاهده پیام‌ها

- تمام پیام‌ها در بخش "پیام‌های محلی" نمایش داده می‌شوند
- جدیدترین پیام‌ها در بالا قرار دارند
- هر پیام شامل عنوان، متن، دسته‌بندی، زمان ارسال و نام فرستنده است

### 3. فیلتر کردن پیام‌ها

- از دکمه‌های دسته‌بندی در بخش فیلتر استفاده کنید
- روی هر دسته کلیک کنید تا فقط پیام‌های آن دسته نمایش داده شود
- برای نمایش همه پیام‌ها روی "همه پیام‌ها" کلیک کنید

### 4. حذف پیام

- روی دکمه "حذف" در انتهای هر پیام کلیک کنید
- پیام به صورت دائمی از localStorage حذف می‌شود

### 5. حذف تمام پیام‌ها

- روی دکمه "پاک کردن همه" کلیک کنید
- تأیید کنید که می‌خواهید تمام پیام‌ها حذف شوند

### 6. فعال‌سازی اعلان‌ها

1. روی دکمه "فعال‌سازی اعلان‌ها" در بالای صفحه کلیک کنید
2. به مرورگر اجازه دهید تا اعلان‌ها را نمایش دهد
3. از این پس برای هر پیام جدید یک نوتیفیکیشن دریافت خواهید کرد

## 💾 ذخیره‌سازی داده‌ها

### localStorage

تمام داده‌ها در localStorage مرورگر ذخیره می‌شوند:

- **کلید ذخیره‌سازی پیام‌ها**: `localBoardMessages`
- **کلید تنظیمات اعلان**: `localBoardNotification`
- **حداکثر تعداد پیام‌ها**: 100 پیام (قدیمی‌ترین پیام‌ها به صورت خودکار حذف می‌شوند)

### پشتیبان‌گیری

برای پشتیبان‌گیری از پیام‌ها:
1. در کنسول مرورگر (F12) دستور زیر را اجرا کنید:
   ```javascript
   copy(localStorage.getItem('localBoardMessages'))
   ```
2. داده‌ها در کلیپ‌بورد کپی می‌شوند

برای بازیابی پشتیبان:
```javascript
localStorage.setItem('localBoardMessages', 'PASTE_YOUR_BACKUP_HERE')
location.reload()
```

## 🔔 سیستم اعلان‌ها

### قابلیت‌ها

- نوتیفیکیشن دسکتاپ برای پیام‌های جدید
- نمایش عنوان و بخشی از متن پیام
- کلیک روی نوتیفیکیشن صفحه را فعال می‌کند
- بستن خودکار نوتیفیکیشن بعد از 5 ثانیه

### نیازمندی‌ها

- مرورگر باید از Notifications API پشتیبانی کند
- کاربر باید دسترسی را تأیید کرده باشد
- در برخی مرورگرها فقط روی HTTPS کار می‌کند

## ⚙️ تنظیمات پیشرفته

### تغییر حداکثر تعداد پیام‌ها

در فایل `assets/js/app.js` خط زیر را تغییر دهید:
```javascript
const CONFIG = {
    MAX_MESSAGES: 100, // تعداد دلخواه خود را قرار دهید
    // ...
};
```

### تغییر زمان نمایش Toast

```javascript
const CONFIG = {
    TOAST_DURATION: 3000, // به میلی‌ثانیه (3000 = 3 ثانیه)
    // ...
};
```

## 🎨 سفارشی‌سازی ظاهر

### تغییر رنگ‌ها

در فایل `assets/css/style.css` متغیرهای CSS را تغییر دهید:
```css
:root {
    --primary-color: #6366f1;  /* رنگ اصلی */
    --secondary-color: #8b5cf6; /* رنگ ثانویه */
    --success-color: #10b981;   /* رنگ موفقیت */
    /* ... */
}
```

### تغییر فونت

اگر می‌خواهید فونت دیگری استفاده کنید:
1. فایل‌های فونت را در پوشه `assets/fonts/` قرار دهید
2. `@font-face` را در `style.css` تغییر دهید
3. مقدار `font-family` را در `body` تغییر دهید

## 🐛 رفع مشکلات رایج

### پیام‌ها ذخیره نمی‌شوند

- بررسی کنید که localStorage مرورگر فعال باشد
- حافظه localStorage ممکن است پر شده باشد (حداکثر 5-10 مگابایت)
- مرورگر را در حالت خصوصی استفاده نکنید

### فونت‌ها درست نمایش داده نمی‌شوند

- مطمئن شوید فایل‌های فونت در مسیر صحیح قرار دارند
- کش مرورگر را پاک کنید (Ctrl + Shift + Delete)
- مسیر فایل‌های فونت در CSS را بررسی کنید

### اعلان‌ها کار نمی‌کنند

- بررسی کنید که دسترسی به نوتیفیکیشن را تأیید کرده‌اید
- در تنظیمات مرورگر دسترسی‌ها را بررسی کنید
- اگر از HTTP استفاده می‌کنید، برخی مرورگرها نوتیفیکیشن را پشتیبانی نمی‌کنند

## 🔒 امنیت و حریم خصوصی

- **بدون سرور**: هیچ داده‌ای به سرور ارسال نمی‌شود
- **ذخیره محلی**: تمام داده‌ها در دستگاه شما ذخیره می‌شوند
- **بدون ردیابی**: هیچ کد ردیابی یا analytics وجود ندارد
- **متن‌باز**: کد کاملاً قابل بررسی است
- **بدون اینترنت**: پس از بارگذاری اولیه نیازی به اینترنت نیست

## 📝 محدودیت‌ها

- اطلاعات فقط در یک مرورگر و دستگاه ذخیره می‌شوند
- WiFi Direct در این نسخه پیاده‌سازی نشده است (نیاز به WebRTC یا سرور)
- پاک کردن cache مرورگر باعث حذف تمام پیام‌ها می‌شود
- محدودیت 5-10 مگابایت برای localStorage
