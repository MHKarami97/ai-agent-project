# دفتر یادداشت آفلاین راست‌چین

این پروژه یک اپلیکیشن یادداشت‌برداری تک‌صفحه‌ای (SPA) است که با HTML, CSS و JavaScript خالص ساخته شده است. تمام داده‌ها در مرورگر ذخیره می‌شوند، پس برای استفاده نیازی به سرور یا بک‌اند نیست. طرح‌بندی کاملاً راست‌چین (RTL) است و فونت فارسی `Vazirmatn` برای تمام متن‌ها، ورودی‌ها و دکمه‌ها استفاده شده.

## ویژگی‌ها

- ورودی عنوان و متن Markdown برای ثبت یادداشت‌های قابل فرمت
- پیش‌نمایش زنده Markdown هنگام تایپ
- قابلیت ایجاد، ویرایش و حذف یادداشت‌ها
- امکان کشیدن و رها کردن یادداشت‌ها برای مرتب‌سازی بصری
- ذخیره‌سازی با `localStorage` برای باقی ماندن یادداشت‌ها بین باز و بسته شدن مرورگر
- طراحی کاملاً ریسپانسیو برای موبایل و دسکتاپ
- آماده میزبانی روی GitHub Pages (فقط فایل‌ها را در شاخه `main` یا `gh-pages` منتشر کنید)

## ساختار پروژه

```
/
├── index.html          # صفحه اصلی کاملاً راست‌چین شده
├── assets/
│   ├── styles.css     # استایل‌های RTL، ریسپانسیو و درگ‌ون‌دراپ
│   └── app.js         # لاجیک یادداشت‌ها (Markdown، Drag&Drop، localStorage)
└── README.md
```

## راه‌اندازی و توسعه

1. مخزن را کلون کنید و وارد پوشه شوید:
   ```powershell
   git clone <repo> my-notes-app
   cd my-notes-app
   ```
2. ساختار فایل‌ها آماده است، نیازی به نصب بسته نیست. کافی است `index.html` را در مرورگر باز کنید یا با افزونه‌ای مثل Live Server اجرا کنید.
3. برای آزمایش Markdown، متن‌های حاوی تیتر (`#`)، لیست، بولت یا لینک تایپ کنید تا در پیش‌نمایش و کارت‌های یادداشت به فرمت HTML تبدیل شوند.
4. برای ری‌استایل یا افزودن فونت، فایل `assets/styles.css` را ویرایش کنید.

## استقرار روی GitHub Pages

1. کد را به یک مخزن GitHub پوش کنید.
2. در بخش تنظیمات (Settings) مخزن، در قسمت Pages، شاخه‌ای که `index.html` در آن قرار دارد را انتخاب کنید (`main` یا `gh-pages`).
3. پس از چند لحظه، صفحه‌ را با آدرس `https://<username>.github.io/<repo>` مشاهده خواهید کرد.

## نکات مهم

- فقط مرورگر مسئول ذخیره‌ی داده‌ها است؛ اگر `localStorage` پاک شود، یادداشت‌ها از بین می‌روند.
- در صورت استفاده از GitHub Pages، مطمئن شوید که `assets/` و فایل JS به درستی از مسیر نسبی `assets/` بارگذاری می‌شوند.
- برای تغییر زبان، فایل HTML را ویرایش کرده و المان‌های متنی را ترجمه کنید، زیرا لاجیک سمت کلاینت در `app.js` مستقل از زبان عمل می‌کند.
