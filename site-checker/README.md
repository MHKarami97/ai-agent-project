# بررسی فیلترینگ و تست DNS سایت‌ها

یک برنامه تحت وب ساده و کارآمد برای بررسی دسترسی به سایت‌ها و تشخیص نوع فیلترینگ (DNS، IP، SNI و غیره). این ابزار به شما کمک می‌کند تا وضعیت دسترسی به سایت‌های مختلف را بررسی کرده و نوع مسدودسازی آن‌ها را شناسایی کنید.

## ویژگی‌ها

- ✅ تست DNS و بررسی دسترسی
- 🔍 تشخیص نوع فیلترینگ (DNS Filter، IP Block، SNI Filter)
- 📊 نمایش جزئیات کامل وضعیت HTTP و HTTPS
- 💾 ذخیره سایت‌های مورد علاقه
- 📜 نگهداری تاریخچه آزمایش‌ها
- 🎨 رابط کاربری زیبا و فارسی
- 📱 طراحی ریسپانسیو برای موبایل
- 🔒 ذخیره‌سازی محلی در مرورگر (بدون نیاز به سرور)

## نحوه استفاده

### راه‌اندازی

1. فایل‌های فونت Vazirmatn را در پوشه `assets/fonts/` قرار دهید:
   - `Vazirmatn-Regular.woff2`
   - `Vazirmatn-Medium.woff2`
   - `Vazirmatn-Bold.woff2`

2. فایل `index.html` را در مرورگر باز کنید یا از یک وب‌سرور ساده استفاده کنید.

### استفاده از برنامه

1. **بررسی سایت:**
   - آدرس سایت مورد نظر را در کادر ورودی وارد کنید (مثال: `google.com`)
   - روی دکمه "بررسی" کلیک کنید
   - نتایج شامل وضعیت دسترسی، نوع فیلتر و جزئیات تست‌ها نمایش داده می‌شود

2. **ذخیره سایت:**
   - پس از بررسی یک سایت، می‌توانید آن را به لیست ذخیره شده اضافه کنید
   - سایت‌های ذخیره شده در بخش "سایت‌های ذخیره شده" نمایش داده می‌شوند

3. **مشاهده تاریخچه:**
   - تمام آزمایش‌های انجام شده به صورت خودکار در تاریخچه ذخیره می‌شوند
   - با کلیک روی هر آیتم، می‌توانید جزئیات آن را مشاهده کنید

4. **مدیریت داده‌ها:**
   - برای پاک کردن سایت‌های ذخیره شده یا تاریخچه، از دکمه‌های مربوطه استفاده کنید

## نحوه کار برنامه

برنامه از تکنیک‌های زیر برای تشخیص فیلترینگ استفاده می‌کند:

1. **تست DNS**: بررسی اینکه آیا DNS سایت قابل دسترس است یا خیر
2. **تست HTTP**: بررسی دسترسی از طریق پروتکل HTTP
3. **تست HTTPS**: بررسی دسترسی از طریق پروتکل HTTPS

بر اساس نتایج این تست‌ها، برنامه نوع فیلترینگ را تشخیص می‌دهد:

- **فیلتر DNS**: اگر DNS سایت پاسخ ندهد
- **فیلتر IP**: اگر DNS کار می‌کند ولی هیچ پروتکلی قابل دسترس نباشد
- **فیلتر SNI/HTTPS**: اگر فقط HTTP قابل دسترس باشد
- **فیلتر نشده**: اگر هر دو پروتکل قابل دسترس باشند

## محدودیت‌ها

به دلیل محدودیت‌های امنیتی مرورگر (CORS):
- برخی تست‌ها ممکن است نتایج دقیق نداشته باشند
- برای نتایج دقیق‌تر، توصیه می‌شود از افزونه مرورگر یا برنامه دسکتاپ استفاده شود

## نکات امنیتی

- تاریخچه و سایت‌های ذخیره شده فقط در دستگاه شما موجود است
