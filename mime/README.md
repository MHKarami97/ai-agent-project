# ژنراتور میم فارسی

یک اپلیکیشن تک‌صفحه‌ای ساده برای ساخت میم‌‌های فارسی که با انتخاب یا بارگذاری تصویر، نوشتن متن بالایی/پایینی و تنظیم رنگ و اندازه متن، خروجی قابل دانلود تولید می‌کند. همهٔ پردازش‌ها در مرورگر انجام می‌شود (بدون بک‌اند) و وضعیت فعلی در حافظهٔ محلی مرورگر ذخیره می‌شود تا در بازگشت مجدد به صفحه آخرین حالت ساخته‌شده بازسازی شود.

## ساختار پروژه

- `index.html` – قالب اصلی RTL و بخش‌های پیش‌نمایش و کنترل‌های کاربر.
- `assets/styles/main.css` – استایل‌های رسپانسیو، لود فونت و حالات موبایل.
- `assets/scripts/app.js` – بارگذاری قالب، رندر روی `<canvas>`، ذخیره/بارگذاری از `localStorage`، و تولید فایل PNG.
- `assets/` – شامل زیربخش‌هایی برای استایل‌ها، اسکریپت‌ها و تصاویر مورد نیاز.

## چطور تستش کنم؟

1. از پوشهٔ پروژه یک سرور استاتیک راه‌اندازی کن:

```powershell
npx http-server -p 4173
```

2. مرورگرت را باز کن و به `http://127.0.0.1:4173` برو.
3. تصویر یا قالب دلخواه را انتخاب کن، متن و رنگ را تنظیم کن و روی "دانلود میم" یا "شروع دوباره" کلیک کن.

برای میزبانی روی GitHub Pages کافی است تمام فایل‌ها را در ریشهٔ شاخهٔ `gh-pages` یا `main` (بسته به تنظیمات) قرار دهی.

## نکات تکمیلی

- وضعیت آخرین میم در `localStorage` نگهداری می‌شود.
- تمام فونت‌ها، منطق و دیتا روی کلاینت اجرا می‌شود؛ نیازی به سرور نداریم.
- طراحی رسپانسیو برای نمایش موبایل و دسکتاپ تنظیم شده است.
