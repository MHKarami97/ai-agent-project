# 📝 راهنمای فونت‌ها

در این پوشه باید فایل‌های فونت Vazirmatn قرار داده شود.

## فایل‌های مورد نیاز:

1. `Vazirmatn-Regular.woff2` - فونت با وزن عادی
2. `Vazirmatn-Bold.woff2` - فونت با وزن ضخیم

## دانلود فونت:

می‌توانید فونت Vazirmatn را از منابع زیر دانلود کنید:
- وب‌سایت رسمی فونت
- مخزن گیت‌هاب فونت
- یا از نسخه‌ای که خودتان دارید استفاده کنید
