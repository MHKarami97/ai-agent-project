# تقسیم‌کننده هزینه گروهی (Frontend-only)

این پروژه یک وب‌اپ ساده است که تماماً در مرورگر اجرا می‌شود و داده‌ها را در IndexedDB ذخیره می‌کند. مناسب میزبانی روی GitHub Pages.

اجرای لوکال (بدون npm):

1. بهترین روش: از یک سرور استاتیک محلی استفاده کنید. دو گزینه ساده توی پوشهٔ پروژه وجود دارد:

   - با Python 3:

     ```powershell
     python -m http.server 8000
     # سپس در مرورگر برو به: http://localhost:8000
     ```

   - یا اجرای اسکریپت PowerShell آماده:

     ```powershell
     .\serve.ps1
     # یا با پارامتر پورت: .\serve.ps1 -Port 3000
     ```

   - یا با کلیک روی `serve.bat` (اگر Python نصب باشد اجرا خواهد شد).

2. پس از اجرای سرور محلی، آدرس http://localhost:8000 را در مرورگر باز کنید.

نکته: باز کردن مستقیم `index.html` با `file://` در برخی مرورگرها باعث خطاهای CORS هنگام بارگذاری ماژول‌ها می‌شود؛ بنابراین از روش سرور استفاده کنید.

دیپلوی:

- کل پوشه را می‌توانید روی GitHub (branch gh-pages یا پوشه docs/) قرار دهید و از GitHub Pages برای میزبانی استفاده کنید.
