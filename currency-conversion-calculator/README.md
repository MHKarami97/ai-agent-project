# 💱 ماشین‌حساب تبدیل ارز

یک ماشین‌حساب پیشرفته و کاربردی برای تبدیل ارزهای معمولی و دیجیتال با استفاده از آخرین نرخ‌های لحظه‌ای.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🌟 ویژگی‌ها

- ✅ **تبدیل ارزهای معمولی**: پشتیبانی از تمامی ارزهای اصلی جهان (دلار، یورو، پوند، ین، و...)
- ✅ **تبدیل ارزهای دیجیتال**: پشتیبانی از محبوب‌ترین ارزهای دیجیتال (بیت‌کوین، اتریوم، بایننس، و...)
- ✅ **نرخ‌های لحظه‌ای**: دریافت آخرین قیمت‌ها از APIهای معتبر
- ✅ **رابط کاربری زیبا**: طراحی مدرن و واکنش‌گرا با انیمیشن‌های روان
- ✅ **پشتیبانی کامل از فارسی**: استفاده از فونت Vazirmatn و طراحی RTL
- ✅ **ذخیره تاریخچه**: ذخیره تبدیل‌های قبلی در مرورگر
- ✅ **کش هوشمند**: کش کردن نرخ‌ها برای کاهش درخواست‌های API
- ✅ **نمایش نرخ‌های محبوب**: نمایش نرخ‌های پرطرفدار در صفحه اصلی
- ✅ **کاملاً کلاینت‌ساید**: بدون نیاز به سرور یا بک‌اند
- ✅ **آماده برای GitHub Pages**: قابلیت هاست رایگان و آسان

## 🚀 نحوه استفاده

### دسترسی آنلاین

می‌توانید مستقیماً از نسخه آنلاین استفاده کنید:
```
https://[your-username].github.io/currency-conversion-calculator/
```

### نصب محلی

1. **دانلود پروژه**:
```bash
git clone https://github.com/[your-username]/currency-conversion-calculator.git
cd currency-conversion-calculator
```

2. **اجرای پروژه**:
   - فایل `index.html` را در مرورگر خود باز کنید
   - یا از یک سرور محلی استفاده کنید:
   ```bash
   # با Python 3
   python -m http.server 8000
   
   # با Node.js (نیاز به نصب http-server)
   npx http-server
   ```

3. **مشاهده در مرورگر**:
   - مرورگر را باز کنید و به آدرس `http://localhost:8000` بروید

## 📋 نحوه کار با برنامه

### تبدیل ساده ارز

1. مقدار مورد نظر را در کادر "مقدار" وارد کنید
2. ارز مبدأ را از منوی "از" انتخاب کنید
3. ارز مقصد را از منوی "به" انتخاب کنید
4. روی دکمه "تبدیل کن" کلیک کنید

### تعویض سریع ارزها

- روی دکمه تعویض (دایره‌ای وسط) کلیک کنید تا ارزهای مبدأ و مقصد جابجا شوند

### مشاهده تاریخچه

- تمام تبدیل‌های شما به صورت خودکار ذخیره می‌شوند
- برای پاک کردن تاریخچه، روی دکمه "پاک کردن همه" کلیک کنید

### نرخ‌های محبوب

- در بخش پایین صفحه، نرخ‌های محبوب به صورت خودکار نمایش داده می‌شوند
- این نرخ‌ها هر 5 دقیقه به‌روزرسانی می‌شوند

## 🔧 فناوری‌های استفاده شده

### Frontend
- **HTML5**: ساختار صفحه
- **CSS3**: استایل‌دهی و انیمیشن‌ها
- **JavaScript (ES6+)**: منطق برنامه و تعاملات

### APIها
- **ExchangeRate-API**: برای دریافت نرخ ارزهای معمولی
  - URL: `https://api.exchangerate-api.com/v4/latest/`
  - رایگان و بدون نیاز به کلید API
  
- **CoinGecko API**: برای دریافت نرخ ارزهای دیجیتال
  - URL: `https://api.coingecko.com/api/v3/simple/price`
  - رایگان و بدون نیاز به کلید API

### فونت
- **Vazirmatn**: فونت فارسی مدرن و زیبا
  - از CDN jsDelivr بارگذاری می‌شود

### ذخیره‌سازی
- **LocalStorage**: برای ذخیره تاریخچه تبدیل‌ها
- **Memory Cache**: برای کش کردن موقت نرخ‌ها

## 💾 ذخیره‌سازی داده‌ها

### LocalStorage
برنامه از LocalStorage مرورگر برای ذخیره تاریخچه تبدیل‌ها استفاده می‌کند:

- **حداکثر تعداد**: 10 مورد آخر
- **فرمت ذخیره**: JSON
- **کلید ذخیره‌سازی**: `currency_conversion_history`

### Cache
برای بهینه‌سازی و کاهش درخواست‌های API:

- **مدت زمان کش**: 5 دقیقه
- **ذخیره در**: حافظه موقت مرورگر
- **پاک‌سازی خودکار**: پس از انقضای زمان
