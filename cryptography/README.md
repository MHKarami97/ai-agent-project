# 🔐 رمزنگاری متن امن

یک اپلیکیشن تحت وب برای رمزنگاری و رمزگشایی متن با استفاده از الگوریتم‌های رمزنگاری قوی. این برنامه برای ارسال امن پیام‌ها از طریق کانال‌های ناامن (مانند پیامک، تلگرام و ...) طراحی شده است.

## ✨ ویژگی‌ها

- 🔒 **رمزنگاری قوی**: استفاده از الگوریتم AES-256-GCM
- 🔑 **PBKDF2**: تولید کلید امن از رمز عبور با 100,000 تکرار
- 🎨 **رابط کاربری زیبا**: طراحی مدرن و کاربرپسند با فونت Vazirmatn
- 📱 **ریسپانسیو**: سازگار با موبایل، تبلت و دسکتاپ
- 🌐 **بدون سرور**: تمام عملیات در مرورگر انجام می‌شود
- 💾 **ذخیره‌سازی محلی**: استفاده از IndexedDB برای ذخیره تاریخچه
- 🔤 **انتخاب کاراکترها**: امکان انتخاب نوع کاراکترهای خروجی (فارسی، انگلیسی، اعداد، علائم)
- 📋 **کپی سریع**: کپی متن با یک کلیک
- 📜 **تاریخچه**: مشاهده و مدیریت متن‌های رمزنگاری شده قبلی

## 🚀 نحوه استفاده

### روش اول: اجرای مستقیم

1. فایل `index.html` را در مرورگر خود باز کنید
2. نیازی به نصب هیچ چیزی نیست!

### روش دوم: اجرا با سرور محلی

اگر می‌خواهید از یک سرور محلی استفاده کنید:

#### با Python:
```bash
# Python 3
python -m http.server 8000

# یا
python3 -m http.server 8000
```

#### با Node.js:
```bash
# نصب http-server
npm install -g http-server

# اجرا
http-server -p 8000
```

سپس به آدرس `http://localhost:8000` بروید.

## 📖 راهنمای استفاده

### رمزنگاری متن

1. به تب **رمزنگاری** بروید
2. متن خود را در کادر "متن اصلی" وارد کنید
3. یک کلید خصوصی قوی (حداقل 8 کاراکتر) تعیین کنید
4. نوع کاراکترهای خروجی را انتخاب کنید:
   - فارسی: متن رمزشده فقط با حروف فارسی
   - انگلیسی: متن رمزشده با حروف انگلیسی
   - اعداد: استفاده از اعداد
   - علائم: استفاده از کاراکترهای خاص
5. روی دکمه "رمزنگاری متن" کلیک کنید
6. متن رمزنگاری شده را کپی کنید یا در تاریخچه ذخیره کنید

### رمزگشایی متن

1. به تب **رمزگشایی** بروید
2. متن رمزنگاری شده را در کادر وارد کنید
3. همان کلید خصوصی را که برای رمزنگاری استفاده کرده‌اید وارد کنید
4. روی دکمه "رمزگشایی متن" کلیک کنید
5. متن اصلی نمایش داده می‌شود

### مدیریت تاریخچه

1. به تب **تاریخچه** بروید
2. تمام موارد ذخیره شده را مشاهده کنید
3. می‌توانید:
   - متن رمزشده را کپی کنید
   - مستقیماً به رمزگشایی بروید
   - موارد را حذف کنید
   - تمام تاریخچه را پاک کنید

## 🔒 امنیت

### الگوریتم‌های استفاده شده:

- **AES-256-GCM**: یک الگوریتم رمزنگاری متقارن بسیار قوی
- **PBKDF2**: برای تبدیل رمز عبور به کلید رمزنگاری با 100,000 تکرار
- **SHA-256**: تابع هش برای PBKDF2
- **Salt تصادفی**: برای جلوگیری از حملات rainbow table
- **IV تصادفی**: برای هر رمزنگاری یک IV منحصر به فرد

### نکات امنیتی:

⚠️ **مهم**: بدون کلید خصوصی، رمزگشایی غیرممکن است!

- کلید خصوصی خود را در جای امنی نگه دارید
- از کلیدهای قوی (حداقل 12 کاراکتر با ترکیب حروف، اعداد و علائم) استفاده کنید
- هرگز کلید خصوصی را همراه با متن رمزشده ارسال نکنید
- کلید را از طریق کانال امن (تماس صوتی، ملاقات حضوری) به گیرنده منتقل کنید
- تمام عملیات در مرورگر شما انجام می‌شود و هیچ داده‌ای به سرور ارسال نمی‌شود
- تاریخچه فقط در مرورگر شما ذخیره می‌شود

## 📝 نمونه استفاده

### سناریو 1: ارسال پیامک محرمانه

1. کاربر A می‌خواهد یک پیام مهم به کاربر B بفرستد
2. کاربر A و B از قبل یک کلید خصوصی را توافق کرده‌اند
3. کاربر A متن را با کلید رمزنگاری می‌کند
4. متن رمزشده را با پیامک ارسال می‌کند
5. کاربر B متن را دریافت و با کلید رمزگشایی می‌کند

### سناریو 2: ذخیره یادداشت‌های محرمانه

1. متن حساس خود را وارد کنید
2. با یک کلید قوی رمزنگاری کنید
3. متن رمزشده را در جای امن (مثلاً Google Keep) ذخیره کنید
4. هر زمان نیاز داشتید، با کلید خود رمزگشایی کنید

## 🤔 سوالات متداول

### آیا این برنامه امن است؟

بله، این برنامه از الگوریتم‌های استاندارد و تایید شده استفاده می‌کند. اما امنیت کامل به انتخاب کلید قوی و محافظت از آن بستگی دارد.

### آیا متن‌های من به سرور ارسال می‌شود؟

خیر، تمام عملیات در مرورگر شما انجام می‌شود. هیچ داده‌ای به سرور ارسال نمی‌شود.

### اگر کلید خود را فراموش کنم چه می‌شود؟

متاسفانه بدون کلید، رمزگشایی غیرممکن است. این یکی از ویژگی‌های امنیتی برنامه است.

### آیا می‌توانم از این برنامه برای ارسال پیام‌های بانکی استفاده کنم؟

بله، اما توصیه می‌شود برای اطلاعات بسیار حساس از روش‌های اضافی امنیتی نیز استفاده کنید.

### چرا باید نوع کاراکترهای خروجی را انتخاب کنم؟

برخی سیستم‌ها (مثل پیامک) ممکن است با برخی کاراکترها مشکل داشته باشند. با انتخاب نوع مناسب، از ارسال صحیح پیام اطمینان حاصل می‌کنید.
