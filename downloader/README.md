# Social Media Downloader (Browser-only)

این یک اپلیکیشن استاتیک است که صرفاً در مرورگر اجرا می‌شود و تلاش می‌کند رسانه (عکس/ویدیو) را از پست‌های عمومی اینستاگرام، تلگرام، توییتر و لینکداین استخراج کند.

نکات مهم:
- برای واکشی HTML از صفحات سرورهای شبکه‌های اجتماعی از پراکسی‌های CORS عمومی استفاده شده — این پراکسی‌ها ممکن است محدود یا لاگ‌گیر باشند. برای حفظ حریم خصوصی پیشنهاد می‌شود یک Cloudflare Worker یا پراکسی کوچک خودتان هاست کنید.
- فقط محتوای عمومی و قانونی را دانلود کنید.

فایل‌ها:
- `index.html` — رابط کاربری
- `assets/css/style.css` — استایل
- `assets/js/app.js` — منطق واکشی، پارس و دانلود
- `assets/css/Vazirmtn.woff2` — فونت پیشنهاد شده (در صورت دلخواه اضافه کنید)

اجرای محلی:
فایل‌ها را روی ریپوزیتوری گیت‌هاب یا یک میزبان استاتیک مثل GitHub Pages آپلود کنید — نیازی به سرور نیست.

توسعهٔ بیشتر:
- اضافه کردن لیست پراکسی بیشتر یا یک دکمهٔ "خود-هاست کردن پراکسی" با نمونه کد Cloudflare Worker
- بهبود استخراج برای تغییرات سایت‌ها
- اضافه کردن تست‌ها و مدریت خطاها
