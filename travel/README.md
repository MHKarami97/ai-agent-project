# Travel Planner

برنامه‌ریز سفر ساده با قابلیت محاسبه فاصله و زمان سفر بین دو شهر و برنامه‌ریزی مسیرها.

## ویژگی‌ها

- طراحی RTL با قلم Vazirmatn در تمام متن‌ها، دکمه‌ها و فیلدها.
- محاسبه‌گر فاصله و زمان سفر بین دو شهر با استفاده از داده‌های آزاد نقشه (OpenStreetMap + سرویس OSRM) و در صورت نیاز الگوریتم کروی داخلی.
- برنامه‌ریز سفر که مسیرها را همراه با مدت، بودجه و یادداشت ذخیره می‌کند.
- داده‌ها با استفاده از `localStorage` مرورگر ذخیره می‌شوند و نیازی به بک‌اند نیست.
- طراحی رسپانسیو بهینه برای دسکتاپ و موبایل.

## استفاده محلی

1. در پوشه پروژه، یک سرور ساده راه‌اندازی کنید (مثلاً Python):

```powershell
cd D:\Local\ai-agent-project\travel
python -m http.server 8000
```

2. مرورگر را باز کنید و به `http://localhost:8000` بروید.
3. تغییرات در HTML/CSS/JS فوراً قابل مشاهده است؛ تنها صفحه را رفرش کنید.

## انتشار روی GitHub Pages

1. مخزن گیت بسازید (اگر وجود ندارد):

```powershell
git init
git add .
git commit -m "initial travel planner"
```

2. مخزن را روی GitHub ایجاد و به آن پوش کنید:

```powershell
git remote add origin https://github.com/<username>/travel-planner.git
git push -u origin main
```

3. در تنظیمات مخزن GitHub، بخش GitHub Pages را باز کرده و شاخه `main` (یا `gh-pages`) را به‌عنوان منبع انتخاب کنید.
4. پس از چند دقیقه، سایت با آدرس `https://<username>.github.io/travel-planner/` در دسترس خواهد بود.

## نحوه کار

- فرم اول (`محاسبه‌گر فاصله`) از شما می‌خواهد مبدا، مقصد و سرعت متوسط را وارد کنید. اپلیکیشن ابتدا تلاش می‌کند فاصله واقعی را از سرویس آزاد OSRM (بر پایه OpenStreetMap) به دست آورد و اگر در دسترس نبود، با الگوریتم کروی داخلی فاصله تخمینی را برآورد می‌کند.
- در هر حالت، نتیجه شامل فاصله، مدت سفر بر اساس سرعت انتخاب شده و منبع داده (OSRM یا محاسبه داخلی) است.
- فرم دوم (`برنامه‌ریز مسیر`) به شما اجازه می‌دهد برنامه‌هایی با عنوان، مدت زمان شبانه، بودجه و یادداشت ثبت کنید. لیست برنامه‌ها در بخش پایینِ فرم نمایش داده می‌شود.
- دکمه "پاک کردن همه" کل برنامه‌ها را از `localStorage` حذف می‌کند.
